# ROKEY 부트캠프 4기
2025.03.10 ~ 2025.09.05 에 진행했던 두산 로보틱스 ROKEY 부트캠프 실습 코드입니다.
M0609, Turtlebot3&4를 이용해 기계 제어를 경험해보는 프로젝트입니다.
약 2개월 동안 총 4개의 프로젝트를 각 2주간 진행하였습니다.

## 협동 1
### 삼겹살 구이 자동화 시스템
두산 M0609 로봇팔의 다양한 움직임을 이용해서 삼겹살을 자동으로 구워주는 시스템을 구축하였습니다. 
고기뒤집기, 물 따르기 등의 동작을 구현하였습니다.
## 협동 2
### LLM & 컴퓨터 비전 기반 요리 보조 로봇
두산 M0609 로봇팔, RGBD 카메라, 마이크를 이용하였습니다. 
Open AI LLM을 이용해 음성을 명령으로 해석하고 로봇을 제어합니다. 
YOLO v8 seg를 사용해 객체를 탐지하고 PCA를 활용해 중간점과 벡터를 구해 안정적으로 잡을 수 있도록 하였습니다. 
MediaPipe의 hand landmark를 활용해 손 동작으로 로봇을 제어할 수 있도록 하였습니다.
## 협동 3
### Turtlebot3를 이용한 차선 유지
Turtlebot3에 바닥을 바라보는 카메라를 설치해 차선 유지를 하며 트랙을 돌게하는 프로젝트입니다.
시뮬레이션(gazebo)으로 우선 구현을 한 뒤 실제 로봇으로 구현하는 디지털 트윈을 경험하였습니다.
## 지능 1
### Aerobot(공항 짐운반 로봇)
Turtlebot4, RGBD 카메라, LiDAR를 사용하였습니다.
RGB카메라로 YOLO를 실행해 객체를 탐지하고 D 카메라를 이용해 좌표값을 구합니다.
이후 SLAM과 Nav2 이용해 해당 좌표로 이동하는 자율주행을 경험하였습니다.
